## Getting Started

### Run Locally

```bash
# Clone the repo
git clone https://github.com/sadykovIsmail/frontend-projects-collection.git
cd frontend-projects-collection

# Navigate to any project
cd 28-memory-card-game

# Install dependencies
npm install

# Start dev server
npm run dev
# → http://localhost:5173
```

### Build for Production

```bash
npm run build
# Output goes to docs/ folder (GitHub Pages compatible)
```

---

## Deployment

All 38 projects are deployed to **GitHub Pages** using the `docs/` folder output strategy. Each project's production build is committed to its respective `docs/` directory and served via GitHub Pages.
