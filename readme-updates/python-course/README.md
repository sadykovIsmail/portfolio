## Learning Concepts

### Classes & Objects
Object-oriented programming fundamentals in Python:
- Class definition and instantiation
- Instance vs class attributes
- `__init__`, `__str__`, `__repr__` dunder methods
- Inheritance and method overriding
- Encapsulation with properties

### Decorators
Higher-order functions that wrap and extend behavior:
- Function decorators (`@decorator`)
- Using `functools.wraps`
- Parameterized decorators
- Practical patterns (logging, timing, caching)

### Inner Functions & Closures
Nested functions and lexical scoping:
- Defining functions inside functions
- Closures and captured variables
- `nonlocal` keyword
- Factory functions

### Graphs
Graph theory and traversal algorithms:
- Adjacency list / adjacency matrix representations
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Path finding

## Getting Started

Python 3.6+ required. No external dependencies needed.
