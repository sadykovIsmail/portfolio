## Overview

A collection of **40 JavaScript projects** built independently while working through [freeCodeCamp's](https://www.freecodecamp.org/) curriculum. These projects were created to deepen practical JavaScript skills beyond certification requirements — covering everything from sorting algorithms to weather apps and productivity tools.

## Getting Started

```bash
# Clone the repository
git clone https://github.com/sadykovIsmail/Java-script.git
cd Java-script

# Open any project
cd 35-sorting-visualizer
open index.html      # macOS
# OR: right-click index.html → "Open with browser"
```

No build tools required — all projects run directly from `index.html`.
